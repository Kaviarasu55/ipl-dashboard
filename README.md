# 🏏 IPL Player Performance Dashboard

An AI-powered interactive IPL analytics dashboard built with Python, Streamlit, and Plotly.

## 🚀 Features
- 📊 **Player Stats** — Heatmap, Radar Chart, year-by-year analysis
- 🏆 **Fantasy Leaderboard** — Batting, Bowling, Allrounder, Overall
- ⚔️ **Player Comparison** — Side-by-side career comparison
- 🤖 **RAG Assistant** — Ask anything about IPL players in plain English
- 🔮 **Prediction** — ML-based performance prediction

## 🛠️ Tech Stack
- Python, Streamlit, Plotly, Pandas, Scikit-learn
- Groq LLM (llama-3.3-70b) for AI features
- TF-IDF + Text-to-Pandas RAG system

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run Home.py
